Hyprland Keybinding Reference

Quick reference for all your Hyprland shortcuts

Applications

⊞ + Enter Terminal (kitty)
⊞ + F File Manager
⊞ + B Firefox Browser
⊞ + / 1Password
⊞ + Space App Menu (rofi)
⊞ + C Clipboard Manager

Window Navigation

⊞ + H/J/K/L Move Focus
⊞ + ⇧ + H/J/K/L Swap Windows
⊞ + Tab Cycle Windows
⊞ + Q Close Window
⊞ + V Toggle Float
⊞ + Z Fullscreen
⊞ + S Toggle Split

Window Resizing

⊞ + - Resize Width (-100)
⊞ + = Resize Width (+100)
⊞ + ⇧ + - Resize Height (-100)
⊞ + ⇧ + = Resize Height (+100)
⊞ + LMB Drag Window
⊞ + RMB Resize Window

Workspaces

⊞ + 1-9,0 Switch to Workspace
⊞ + ⇧ + 1-9,0 Move Window to WS
⊞ + Scroll ↑ Previous Workspace
⊞ + Scroll ↓ Next Workspace

System & Power

⊞ + Esc Lock Screen
⊞ + ⇧ + Esc Suspend
⊞ + Alt + Esc Exit Hyprland
⊞ + Ctrl + Esc Reboot
⊞ + ⇧ + Ctrl + Esc Shutdown

Media & Volume

🔊 Volume Up 5%
🔉 Volume Down 5%
🔇 Toggle Mute
🎤 Toggle Mic Mute
⏯ Play/Pause
⏭/⏮ Next/Previous

Brightness

🔆 Brightness Up 5%
🔅 Brightness Down 5%

Screenshots

Print Region Screenshot
⇧ + Print Window Screenshot
Ctrl + Print Output Screenshot
⊞ + Print Color Picker
Legend: ⊞ = Super/Windows Key | ⇧ = Shift | Alt = Alt Key | Ctrl = Control Key
H/J/K/L = Vim-style direction keys (Left/Down/Up/Right)
